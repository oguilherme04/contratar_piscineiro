Projeto de Conclusão de Curso

Para o desenvolvimento do sistema web, utilizamos:

  • Framework: Django
  • Banco de dados: SQLite3

O Django foi escolhido por sua robustez, segurança e facilidade de desenvolvimento de aplicações web.

Frontend

Para a interface do usuário, utilizamos:

  • Framework CSS: Bootstrap
  • Templates: HTML5 com Django Template Language
  • Ícones: Font Awesome

O Bootstrap foi selecionado por oferecer componentes responsivos e modernos, garantindo uma boa experiência do usuário em diferentes dispositivos.

Objetivo do Projeto

Este sistema foi desenvolvido com o objetivo de oferecer uma solução acessível e eficiente para o monitoramento da qualidade da água em piscinas, através da medição precisa do pH e temperatura.

Diferente das soluções convencionais do mercado, que muitas vezes são de alto custo e complexidade operacional, nosso sistema oferece:

  • Medições precisas de pH e temperatura
  • Interface web intuitiva e responsiva
  • Recomendações personalizadas para manutenção
  • Alertas para variações fora dos parâmetros ideais
  • Histórico de medições para acompanhamento
  • Custo acessível de implementação

O sistema integra hardware e software para fornecer uma solução completa que contribui para a segurança dos usuários, economia de produtos químicos e preservação dos recursos hídricos.
